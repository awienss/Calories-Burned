# Calories Burned Calculator

The Calories Burned Calculator is a web application that estimates the calories burned for a variety of activities based on user inputs such as weight, duration, and activity type. Built using Flask, the app interacts with the API Ninjas Calories Burned API to deliver accurate, real-time data to users.  

# Features

- Search for calories burned across hundreds of activities.  
- Customize inputs for weight (lbs) and duration (minutes).  
- Intuitive and responsive user interface.  
- Dynamic error handling and validation for a seamless user experience.   
- API: https://api.api-ninjas.com/v1/caloriesburned

# Usage  

1. Clone this repository: git clone https://github.com/yourusername/calories-burned-calculator.git  

2. Navigate to the project directory: cd calories-burned-calculator

3. Obtain a free API key from API Ninjas and add your API key to the app.py file.

4. Run the Flask application (python app.py) and access the app in your browser.
